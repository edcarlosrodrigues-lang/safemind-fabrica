## 🎯 O Problema
Líderes de indústria não tinham visibilidade em tempo real sobre o bem-estar das equipes nos 3 turnos, dificultando ações preventivas de saúde mental e SST.

## ✅ A Solução
SafeMind é uma plataforma web que coleta autoavaliações diárias dos colaboradores e exibe dashboards com gráficos para Gestores, Supervisores e Coordenadores de cada turno.

---

## 🚀 Funcionalidades

| Perfil | Acesso |
| --- | --- |
| **Colaborador** | Formulário diário de humor, energia e alertas. 100% anônimo. |
| **Gestor** | Dashboard geral com gráficos, filtros por data/turno e export CSV. |
| **Supervisão/Coordenação** | Acesso restrito aos dados do seu turno específico. |

### Destaques Técnicos
- ✅ Login por turno sem senha - 9 perfis diferentes
- ✅ Dados em tempo real com Firebase Realtime Database  
- ✅ Gráficos dinâmicos com Chart.js
- ✅ Exportação de relatórios em CSV
- ✅ 100% Responsivo para celular
- ✅ Deploy automático CI/CD via GitHub + Netlify
